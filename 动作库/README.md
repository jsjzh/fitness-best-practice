# 动作库

因为工作原因，笔者每天健身的时间不算很多，早上八点八点半到公司，晚上七八点下班，健身的时间只有一个小时，为了利用最少的时间来达到最好的健身效果，建议各位男生将组间休息时间控制在 30 秒以内，尽量不要超过 1 分钟，短时间内对一个部位狂轰乱炸可以加强增肌效果。

如果你是女孩子，因为女生和男生训练的侧重点不同，所以笔者建议动作库里的动作以小重量大组数为主。

另外，对于每周应该如何安排自己的训练计划，哪些肌肉群可以第一天狂轰乱炸之后第二天继续训练，哪些肌肉群一周不宜练习太多次不然容易受伤，我会整理一篇文章，有需要的人可以看这里。
